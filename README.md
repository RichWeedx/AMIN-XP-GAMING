# AMIN-XP-GAMING

React + Vite project setup so you can import your own React website files here.

## Quick start

```bash
npm install
npm run dev
```

## Import your website files

1. Copy your React website files into `/tmp/workspace/RichWeedx/AMIN-XP-GAMING/src`.
2. Update `/tmp/workspace/RichWeedx/AMIN-XP-GAMING/src/main.jsx` if your app entry file is different.
3. Replace `/tmp/workspace/RichWeedx/AMIN-XP-GAMING/src/App.jsx` with your own root component.

## Validate

```bash
npm run lint
npm run build
```
